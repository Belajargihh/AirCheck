Tentang Kami

Apa itu AirCheck?

Sistem pakar berbasis kecerdasan buatan untuk menganalisis kualitas udara di Kota Sorong.

Teknologi AI

Menggunakan Natural Language Processing (NLP) dan algoritma Naive Bayes untuk menganalisis kondisi udara.

Input Natural

Cukup deskripsikan kondisi dengan bahasa sehari-hari, sistem akan memahami dan mengklasifikasikan.

Hasil Akurat

Memberikan probabilitas dari setiap kategori kualitas udara beserta saran kesehatan.

Cara Kerja

Bagaimana Sistem Bekerja?

Proses analisis dilakukan dalam 3 tahap sederhana

1

NLP Processing

Sistem membaca input Anda, melakukan Case Folding, Tokenizing, Stopword Removal, dan Stemming untuk mengekstrak kata kunci penting.

2

Naive Bayes Classification

Kata kunci yang diekstrak diproses dengan algoritma Naive Bayes untuk menghitung probabilitas setiap kategori kualitas udara (Baik, Sedang, Tidak Sehat).

3

Hasil & Rekomendasi

Sistem memilih kategori dengan probabilitas tertinggi dan memberikan saran kesehatan yang relevan berdasarkan hasil klasifikasi.

FAQ

Pertanyaan yang Sering Diajukan

Temukan jawaban untuk pertanyaan umum seputar AirCheck

AirCheck adalah sistem pakar berbasis web yang menganalisis kualitas udara di Kota Sorong menggunakan teknologi Natural Language Processing (NLP) dan algoritma Naive Bayes. Sistem ini dikembangkan oleh Kelompok 8 Teknik Informatika UNIMUDA Sorong.

Cukup deskripsikan kondisi udara di sekitar Anda dengan bahasa sehari-hari di kolom input. Contoh: "Udara terasa sesak, ada kabut tebal, dan berbau asap." Sistem akan otomatis menganalisis dan memberikan hasil klasifikasi kualitas udara.

Terdapat 3 kategori kualitas udara:
• Baik - Udara bersih dan aman untuk semua aktivitas.
• Sedang - Kualitas udara cukup, perlu perhatian bagi kelompok sensitif.
• Tidak Sehat - Udara berbahaya, hindari aktivitas di luar ruangan.

Naive Bayes adalah algoritma machine learning yang digunakan untuk klasifikasi teks. Algoritma ini menghitung probabilitas setiap kategori berdasarkan kata-kata yang muncul dalam input, lalu memilih kategori dengan probabilitas tertinggi.

Tidak. AirCheck tidak menyimpan data input dari pengguna. Semua proses analisis dilakukan secara real-time dan data tidak disimpan di server setelah sesi berakhir.

AirCheck dikembangkan oleh Kelompok 8 Program Studi Teknik Informatika, Universitas Pendidikan Muhammadiyah (UNIMUDA) Sorong sebagai proyek mata kuliah tahun 2026.
